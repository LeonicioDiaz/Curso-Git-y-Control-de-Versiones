Bienvenidos 👋

Este sitio web es el resultado del trabajo colaborativo de los estudiantes del curso “Git y Control de Versiones”. Cada participante creó su propia rama, realizó cambios y contribuyó al repositorio principal mediante Pull Requests.

El objetivo fue aprender de forma práctica el flujo completo de Git: commits, ramas, merges y colaboración en GitHub.

👥 Participantes

Cada participante agregó su perfil o una breve descripción en la carpeta /participantes.
